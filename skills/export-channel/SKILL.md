# export-channel

한국 제조 자영업·중소기업의 해외 B2B 판로 개척 가이드. 시장 선정부터 통관·정부 지원까지 단계별로.

## 발동 조건

- "해외에 팔고 싶어", "수출하려면 뭐부터", "바이어 어떻게 찾아"
- "인코텀즈", "FOB/CIF 차이", "L/C가 뭐야", "수출 서류"
- "무역 시작", "OEM 해외 납품", "전시회 나가야 하나"
- 국내 판매 중인 제품을 해외로 확장하려는 의사결정 단계

## 0단계: 진단 (AskUserQuestion)

코드 짜지 말고 먼저 묻는다. 답에 따라 이후 단계 깊이를 조절한다.

| 질문 | 왜 중요 |
|---|---|
| 제품군·HS코드 알고 있나 | 관세·인증·통관 전부 HS코드 기준 |
| 현재 수출 경험 (전무/간접/직접) | 초보면 간접수출(종합상사·벤더)부터 권장 |
| 타깃 지역 후보 있나 | 없으면 1단계 시장 선정부터 |
| MOQ·단가·생산 capa | 바이어 발굴·전시회 적합성 판단 |
| 인증 보유 (CE/FDA/FCC 등) | 미보유면 시장 진입 자체 불가 영역 존재 |

## 1단계: 시장 선정 → simon-research 핸드오프

직접 리서치하지 말고 `simon-research`로 넘긴다. 넘길 때 리서치 질의를 구체화:

- 타깃 후보국 3개 + 해당국 수입 규모·관세율·필수 인증
- 경쟁 제품 현지 가격대, 주요 유통 채널
- FTA 활용 가능 여부 (한-EU, 한-미, RCEP 등 → 관세 절감)

데이터 출처(WebSearch/WebFetch로 보강):
- 무역통계: K-stat (kstat.kita.net), UN Comtrade (comtrade.un.org)
- 관세·FTA: 관세청 FTA 포털 (customs.go.kr), 통합무역정보 TradeNAVI (tradenavi.or.kr)

## 2단계: 바이어 발굴

| 채널 | 성격 | 비용 | 적합 |
|---|---|---|---|
| **KOTRA 지사화·바이어매칭** | 정부 무료/저가 지원, 현지 지사 대행 | 낮음 | 초보·중소기업 1순위 |
| **해외 전시회 (KOTRA 단체관)** | 대면 상담, 신뢰 구축 | 중~높음 | 샘플 있는 제조사 |
| **tradeKorea** (KITA 운영) | 국내 B2B 마켓, 인콰이어리 수신 | 무료 | 진입 초기 |
| **Alibaba.com** | 글로벌 B2B, 트래픽 최대 | 멤버십 유료 | 표준 소비재 |
| **Buy Korea** (KOTRA) | 정부 운영 B2B 매칭 | 무료 | 정부 지원 연계 |
| **GlobalSources / EC21** | 산업재·전자 강세 | 유료 | 부품·산업재 |

원칙: 초보는 KOTRA 무료 지원 + tradeKorea/Buy Korea로 시작 → 인콰이어리 검증되면 전시회·Alibaba 유료 투자.

## 3단계: 가격 + 인코텀즈 (Incoterms 2020)

견적은 반드시 인코텀즈 조건을 명시한다. "USD 10/EA" 가 아니라 "USD 10/EA FOB Busan".

| 조건 | 위험·비용 분기점 | 매도인(수출자) 부담 | 운송수단 |
|---|---|---|---|
| **EXW** | 공장 인도 | 최소 (공장 출고만) | 모든 |
| **FCA** | 지정장소 운송인 인도 | 수출통관까지 | 모든 |
| **FOB** | 선적항 본선 적재 | 본선 적재까지 | 해상 전용 |
| **CFR** | (FOB+해상운임) | 운임 부담, 위험은 적재시 이전 | 해상 전용 |
| **CIF** | (CFR+보험) | 운임+보험, 위험은 적재시 이전 | 해상 전용 |
| **CPT/CIP** | 컨테이너·복합운송용 CFR/CIF | CIP는 보험 ICC(A) 권장 | 모든 |
| **DAP** | 목적지 도착 (하역 전) | 목적지까지 운송 | 모든 |
| **DDP** | 목적지 인도 (수입통관·관세 포함) | 최대 (전부 부담) | 모든 |

핵심 함정:
- 컨테이너 화물은 FOB/CFR/CIF 아닌 **FCA/CPT/CIP** 가 정석 (위험 이전 시점 정확).
- 초보 수출자 권장 출발선: **FOB** (해상) 또는 **FCA** (컨테이너). DDP는 현지 세무 리스크 커서 피한다.
- 원가표: 제조원가 + 내륙운송 + 수출통관 + (조건별) 해상운임 + 보험 + 마진.

## 4단계: 결제조건 (수출자 리스크 순)

| 방식 | 수출자 리스크 | 설명 | 권장 상황 |
|---|---|---|---|
| **선급금 T/T (Advance)** | 최저 | 선적 전 송금 수취 | 신규·소액 거래 이상적 |
| **L/C (신용장)** | 낮음 | 개설은행이 지급 보증 | 중대형·신규 바이어 |
| **D/P** (지급인도) | 중간 | 서류=대금 동시, 은행 추심 | 어느 정도 신뢰 |
| **D/A** (인수인도) | 높음 | 외상, 만기 후 지급 | 거래 안정기만 |
| **후급 T/T (Open Account)** | 최고 | 선적 후 송금 약속 | 장기 신뢰 거래만 |

원칙:
- 첫 거래는 **선급 T/T 30% + 잔금 선적 전** 또는 **L/C at sight**.
- D/A·후급은 반드시 **무역보험(K-SURE) 단기수출보험**으로 미수금 헤지.
- L/C 받으면 조건(선적기일·서류·불일치) 꼼꼼히 검토 — 사소한 불일치(discrepancy)로 지급 거절되는 게 가장 흔한 사고.

## 5단계: 수출 서류

| 약어 | 서류 | 발행 주체 | 용도 |
|---|---|---|---|
| **C/I** | Commercial Invoice (상업송장) | 수출자 | 대금·통관 기준 |
| **P/L** | Packing List (포장명세서) | 수출자 | 화물 내역·검수 |
| **B/L** | Bill of Lading (선하증권) | 선사 | 화물 권리증서 (해상) |
| **AWB** | Air Waybill | 항공사 | 항공 화물 (비유통증권) |
| **C/O** | Certificate of Origin (원산지증명) | 상공회의소/세관 | FTA 관세 혜택·통관 |
| **L/C** | 신용장 사본 | 개설은행 | 결제 조건 |
| **Ins.** | 보험증권 | 보험사 | CIF/CIP 시 필수 |

원산지증명(C/O)은 FTA 특혜관세의 핵심 — 대한상공회의소(cert.korcham.net) 또는 세관 발급. 발급 안 하면 바이어가 관세를 더 내고, 가격경쟁력이 죽는다.

## 6단계: 물류·통관

1. **포워더(국제물류주선업체) 선정** — 운임 견적 2~3곳 비교, FCL(만재)/LCL(혼재) 결정.
2. **수출통관** — 관세청 UNI-PASS 신고, 수출신고필증 수령. 관세사 대행이 일반적.
3. **선적** — 부킹 → 컨테이너 반입 → On Board → B/L 수령.
4. **사후관리** — 수출신고필증은 영세율 부가세 신고·관세 환급 근거이므로 보관.

체크: 전략물자(이중용도) 해당 여부 → 해당 시 산업부 수출허가 필요 (yestrade.go.kr).

## 7단계: 정부 지원 (한국)

| 기관/제도 | 지원 내용 | 링크 |
|---|---|---|
| **KOTRA** | 지사화, 바이어매칭, 전시회 단체관, 해외시장 조사 | kotra.or.kr |
| **무역협회(KITA)** | tradeKorea, 무역실무 교육, 무역애로 상담 | kita.net |
| **무역보험공사(K-SURE)** | 단기수출보험(미수금 헤지), 환변동보험 | ksure.or.kr |
| **중소벤처기업진흥공단** | 수출바우처, 수출 인큐베이터 | kosmes.or.kr |
| **수출바우처** | 인증·마케팅·통번역 등 메뉴판식 지원 | exportvoucher.com |
| **관세청 FTA 포털** | FTA 활용·원산지 컨설팅 | customs.go.kr |

원칙: 초보 수출은 **KOTRA 지사화 + 수출바우처** 조합으로 비용 최소화. D/A·후급 거래는 **K-SURE 단기수출보험** 필수 연계.

## 산출물

대화로 다음을 정리해 제시 (파일 생성 불필요):
- 진단 요약 (제품·경험·타깃·인증 갭)
- 추천 진입 시나리오 (간접 vs 직접, 1순위 채널)
- 권장 인코텀즈 + 결제조건 조합 + 근거
- 필요 서류 체크리스트
- 연계할 정부 지원 제도 2~3개
