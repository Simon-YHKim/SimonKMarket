# /community-marketing — 커뮤니티 마케팅 (정직·공시·휴먼승인)

국가별 실제 커뮤니티에 톤을 맞춰 대규모로 도달하되, **들켜서 밴당하지 않고 평판을 키우는** 방식으로.

## 0. 협상 불가 원칙 (위반 시 중단)
1. **가짜 리뷰·위장 금지** — 무관한 일반 사용자인 척 리뷰/후기를 남기지 않는다. 너는 "만든 사람/관계자"다. astroturfing은 **불법**(US FTC 가짜리뷰 규정 2024 · 한국 표시광고법 대가성 표시 · EU UCPD)이자 커뮤니티 ToS 위반 → 적발 시 앱·계정 영구 차단·평판 붕괴.
2. **공시 강제** — 제작자/광고/이해관계를 명시(예: "제가 만든 앱이에요", "[광고]"). 플랫폼·법규가 요구하는 라벨 포함.
3. **커뮤니티 규칙 준수** — 각 커뮤니티의 자기홍보 정책을 먼저 읽는다. 금지면 홍보글 대신 가치제공 참여 또는 스킵. 스팸·도배·다중계정·추천수 조작 금지.
4. **휴먼 승인 + 본인 계정** — 스킬은 **초안·계획**까지. 게시는 사용자가 본인 인증 계정으로 승인 후 진행(은밀한 봇·가짜계정 금지).

## 1. 의도 진단
`AskUserQuestion`: 어떤 앱(가치·핵심기능·이미지·링크) / 타깃 국가·언어 / 목표(인지·가입·피드백) / 가진 자산(스크린샷·데모·계정).

## 2. 커뮤니티 발굴 (국가별)
WebSearch/WebFetch로 타깃별 실제 커뮤니티를 찾고 **각각의 규칙·성격·말투를 읽는다**:
- 글로벌: Reddit(관련 서브), ProductHunt, IndieHackers, Hacker News, Discord, 전문 포럼
- 한국: 커뮤니티(클리앙·디시·뽐뿌·웃대·블라인드 등)·네이버 카페·오픈채팅·태그 기반 인스타/스레드
- 일본: 5ch·Note·Qiita 등 / 기타 국가별 상응 커뮤니티
각 커뮤니티 분류: 자기홍보 허용? 홍보 요일/태그 있음? 가치제공만 허용? 톤(전문/캐주얼/밈/반말/존댓말)?

## 3. 커뮤니티별 메시지 각색
각 커뮤니티의 **성격·분위기·말투**에 맞게 앱의 가치·기능·이미지를 각색하되 **정직한 제작자 글 또는 가치제공 콘텐츠**로:
- "제가 ~문제를 풀려고 만들었어요" 메이커 스토리 / 해당 커뮤니티에 유용한 정보 + 자연스러운 소개
- `human-voice-guard`(Core) 적용 — AI 티 없이 자연스럽되 **진실만**. 과장·허위 효능 금지.
- 공시 문구 삽입. 커뮤니티 톤에 맞춘 제목·본문·이미지 캡션.

## 4. 컴플라이언스 게이트 (게시 전 자동 점검)
- [ ] 공시 있음? [ ] 신분 정직(제작자)? [ ] 이 커뮤니티가 자기홍보 허용? [ ] 규칙(요일/태그/횟수) 준수? [ ] 허위·과장 없음?
- 가짜 리뷰/위장/조작 감지 시 **하드 차단** + 대안(가치제공 글) 제시.

## 5. 게시 계획 (휴먼 승인)
커뮤니티별 초안 + 최적 게시 시점 + 규칙 메모를 표로 제시 → 사용자 승인. 승인 시 사용자 본인 계정으로 게시(원하면 `web-publisher`로 본인 계정 게시 보조 — 공시·휴먼인루프 유지). 일정 분산(동시 도배 금지).

## 6. 추적·참여·알림
게시처·반응(댓글·질문) 추적 → 질문엔 **정직하게** 응대(제작자로서) → 사용자에게 요약 알림(어디 올림·반응·다음 행동). 부정 피드백은 `feedback-and-review-collector`(예정)/제품 개선으로 연결.

## Boundaries
가짜 리뷰·sockpuppet·추천수/평점 조작·은밀한 자동 도배 = **하지 않음**(불법·밴·평판 리스크). 이건 "정직한 커뮤니티 마케팅 + 톤 매칭 + 운영 자동화"이지 여론조작이 아니다.
